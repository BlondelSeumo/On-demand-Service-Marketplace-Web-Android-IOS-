Installation Guide

To Install this App please follow the following easy steps

Step 1: Database Creation
  • Create database on your server
    • For example: Go to the MySQLWizard, and then Create a below details:
    • Database Name: You can set as per your need.
    • Database User: You can set as per your need.
    • Databased Password: You can set as per your need.
  • Once you have created the database, Upload your mysql file
Step 2: Imports data tables
  • After create the database and import the truelysell.sql file
  • truelysell.sql we have attached download file
Step 3: Upload to server
  • Upload the files to your webserver via FTP or Cpanel
  • Extract your files

  • Log into your cPanel, navigate to the Files section and click the File Manager icon:

  • You will be able to see currently assigned permissions in the right-hand column called Permissions: (0755 is default for all files.)

  • After upload the project your files will be listed, there you can see the file permission

  • Give the necessary permissions and update it

    The permissions are

      There are three types of access:
    • read: the file can be only read
    • write: the file can be edited
    • execute: the file can be executed as a program
      • There are three types of user groups, these access types can be applied to:
    • owner: the owner of the file
    • group: other files which are in the same folder or group
    • world: anyone else
      • The access level is defined in numbers:
    • 0 - no access to the file
    • 1 - execute only
    • 2 - write only
    • 3 - write and execute
    • 4 - read only
    • 5 - read and execute
    • 6 - read and write
    • 7 - read, write and execute (full permissions)
Step 4: Database Connection Settings
  • Go to the Folder which is yourdomain.com/project/application/config/database.php
  • Change your database details which you have created which means(step 1)

  • For your refrence please see the below:

  • What computer/server the database is located on (hostname - server name (www.xyz.com));

    What username to connect to the database with (set user name which we set for database);

    The password associated with that username (set password which we set for database username);

    The name of the database((set database name which we create for database)).

Step 5 : How to import the DB files
  • Go to your phpmyadmin panel through cpanel dashboard

  • Go to the Menu and click the import option

  • Go to the database list and select your db

  • In the import menu you can see the file upload option , In that you have to upload the ".sql" file and then click "Go" button

  • After Clicking the "Go" button your database will be loaded.

Requirements

    Software Requirements

    You need to be running PHP 5.4+ on a web server. PHPKB knowledge base software has been deployed successfully on both Apache and IIS web servers and tested with MySQL, MariaDB and MSSQL Server databases at the backend.
  • Operating System: Linux, Unix, Windows
  • Web Server: Apache Web Server, LigHTTPD, IIS (with ISAPI_Rewrite installed)
  • PHP Version:
    For MySQL & MariaDB Editions, PHP 5.4 or above with PHP XML extension enabled. For SQL Server Editions, PHP 5.4 or above with PHP XML extension enabled and Microsoft SQL Server Driver for PHP
  • Database: MySQL 5.1 or above, MariaDB 10.0 or above, Microsoft SQL Server 2005 or above.
  • Web Browser: All industry standard web browsers (Internet Explorer, Mozilla Firefox, Google Chrome, Apple Safari)

    • Hardware Requirements

  • Disk Space : The actual disk space for your knowledge base could easily be 500MB or more (exclusive of database content, media, backups and other files).
  • To host the software in online ,We defenitely recommended the hosting platform with (Windows, Mac or Linux)
  • Your Hosting Server should be accomplish with the above software and hardware requirements
